# 50. Software Admin Overview & Architecture

## Portal Administrasi Terpusat (Web / Desktop)
Software Admin adalah sistem terpisah yang mengelola seluruh Master Data dan Administrasi Akademik Pesantren:
- **Master Data**: Santri, Alumni, Mustahiq, Munawwib, Pegawai, Orang Tua, Wali, Kelas, Tingkatan, Kamar, Asrama, Kitab, Mapel, Kalender Akademik, Tahun Ajaran, Semester, Hari Libur, Jadwal, Pengumuman, Agenda.
- **Akademik**: Absensi, Nilai, Hafalan, Prestasi, Pelanggaran, Catatan, Kenaikan Kelas, Kelulusan, e-Raport, e-Ijazah, e-Sertifikat.
- **Monitoring & Laporan**: Dashboard grafik, laporan PDF & Excel.
- **Administrasi & System Governance**: User, Role, Permission, RBAC Matrix, Audit Log, Activity Log, Backup, Restore, System Setting, Feature Flag.
