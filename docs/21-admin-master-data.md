# 21. Admin Master Data - Spesifikasi Data Utama

## Cakupan Data Utama
- **Data Santri Putri**: NISN, Nama Lengkap, Tempat/Tgl Lahir, Alamat, Wali, Status Aktif.
- **Data Alumni**: Riwayat kelulusan santri, angkatan, dan domisili alumni.
- **Data Guru (Mustahiq/Munawwib)**: NIPG, Spesialisasi Kitab, Tingkatan Mengajar.
- **Data Kebencanaan & Asrama**: Data Kamar, Asrama, dan Tingkatan Madrasah.
- **Data Akademik**: Kitab/Mata Pelajaran, Kalender Akademik, Jadwal Mengajar, Hari Libur.
