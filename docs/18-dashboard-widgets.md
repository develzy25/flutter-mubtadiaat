# 18. Reusable 3D Dashboard Widget Catalog

## Spesifikasi Widget Reusable (Informatif & Context-Aware)
1. **Greeting Widget**: Profile avatar 3D + Nama Ustadzah + Subtitle peran/kelas binaan.
2. **Profile Widget**: Ringkasan status aktif & info kontak.
3. **Calendar Widget**: Widget tanggal 3D Neumorphic biru (*Rabu, 14 Mei 2025*).
4. **Schedule Widget**: Timeline Jadwal Mengajar + Badge status (*Sedang Berlangsung / Akan Datang*).
5. **Attendance Widget**: Donat/Progress bar ringkasan presensi santri harian.
6. **Statistics Widget**: Card statistik 3D timbul (*Kelas, Jadwal, Absensi, Penilaian*).
7. **Announcement Widget**: Feed pengumuman resmi madrasah.
8. **Agenda Widget**: Pengingat kegiatan madrasah mendatang.
9. **Recent Activity Widget**: Log aktivitas pencatatan presensi/nilai terakhir.
10. **Quick Action Widget**: Grid tombol pintas 3D Neumorphic (*Kelas, Kehadiran, Hafalan, E-Raport*).
11. **Performance Widget**: Progress bar capaian kurikulum & target hafalan.
12. **Summary Widget**: Ringkasan eksekutif statistik santri per tingkatan.
