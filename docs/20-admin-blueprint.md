# 20. Software Admin Blueprint - Portal Administrasi

## Deskripsi Portal Web Admin
Portal Web Admin e-Mubtadi'aat dirancang untuk layar desktop bagi petugas sekretariat, bendahara, dan administrator pusat Pondok Pesantren Hidayatul Mubtadi'at Lirboyo.

## Cakupan Sistem Admin
1. **Master Data Management**: Data Santri, Alumni, Mustahiq, Pegawai, Orang Tua, Kelas, Kitab, & Jadwal.
2. **Akademik & Raport**: Pengelolaan nilai semester, e-raport, e-ijazah, kenaikan kelas, dan kelulusan.
3. **Laporan & Export**: Cetak dokumen resmi dalam format PDF & Excel.
4. **Keuangan & Syahriah**: Pencatatan administrasi keuangan dan pembayaran santri.
5. **System Settings & Governance**: Role-based access control (RBAC), backup/restore, dan audit trail log.
