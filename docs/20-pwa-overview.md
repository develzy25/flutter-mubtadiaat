# 20. PWA Application Architecture & Operational Scope

## Batasan Operasional PWA Mobile
Aplikasi PWA khusus untuk operasional harian di lapangan:
- **TIDAK ADA** role Admin.
- **TIDAK ADA** pengelolaan master data (Santri, Guru, Kelas, Mapel).
- **TIDAK ADA** setting database / konfigurasi sistem.
- **TIDAK ADA** fungsi backup dan restore.
- **TIDAK ADA** CRUD master data.
PWA hanya difokuskan pada presensi harian, penilaian, hafalan, e-raport, e-ijazah, dan monitoring.
