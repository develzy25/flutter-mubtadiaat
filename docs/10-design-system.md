# 10. Design System Specification - 3D Neumorphism & Luxury Soft UI

## Prinsip Desain
Seluruh aplikasi wajib menggunakan gaya tampilan **Luxury 3D Neumorphism, Soft UI, Glassmorphism, dan Floating Cards**. 
Dilarang menggunakan Material Design default, Bootstrap, atau Admin Template standar.

## Token Warna & Surface
- **Surface Background**: `#F0F4F8`
- **Primary Brand Color**: `#1E40AF` (Lirboyo Deep Blue)
- **Secondary Color**: `#93C5FD` (Soft Sky Blue)
- **Dark Neutral Text**: `#1E293B` (Slate 800)
- **Light Neutral Text**: `#64748B` (Slate 500)
- **Success Color**: `#86EFAC` (Emerald Green)
- **Warning Color**: `#FDBA74` (Warm Amber)
- **Danger Color**: `#FCA5A5` (Rose Red)
