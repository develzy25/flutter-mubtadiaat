# 22. Admin Akademik - Modul Nilai & E-Raport

## Fitur Utama
1. **Rekapitulasi Absensi**: Persentase kehadiran santri per semester.
2. **Pengelolaan Nilai & E-Raport**:
   - Penggabungan nilai harian, ujian pertengahan semester, dan ujian akhir semester.
   - Generasi PDF E-Raport resmi Pondok Pesantren Hidayatul Mubtadi'at Lirboyo.
3. **Kenaikan Kelas & Kelulusan**:
   - Pemrosesan massal kenaikan tingkat dan kelulusan santri.
   - Generasi E-Ijazah dan E-Sertifikat.
