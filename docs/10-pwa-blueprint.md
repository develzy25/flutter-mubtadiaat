# 10. PWA Blueprint - Aplikasi Pengajar

## Deskripsi Aplikasi PWA
Aplikasi Progressive Web App (PWA) e-Mubtadi'aat dirancang untuk penggunaan mobile harian para ustadzah/pengajar di Pondok Pesantren Hidayatul Mubtadi'at Lirboyo.

## Fitur Utama PWA
1. **Dashboard 3D Neumorphism**:
   - Kartu sapaan ustadzah, widget kalender 3D, ringkasan sesi mengajar, dan jadwal harian.
2. **Manajemen Kehadiran Santri**:
   - Input presensi kelas cepat (Hadir, Sakit, Izin, Alpha).
3. **Penilaian & E-Raport**:
   - Input nilai ujian semester & cetak rekapitulasi e-raport.
4. **PWA Standalone & Offline Mode**:
   - Dapat diinstal ke Home Screen HP dengan ikon 3D Lirboyo dan bekerja offline via Service Worker.
