# 06. Definisi Peran Resmi (Role Mapping)

## Pemetaan Peran PWA (Resmi 3 Peran Operasional)

### 1. Mustahiq (Wali Kelas / Pengajar Utama)
- **Tanggung Jawab**: Pengelolaan kelas binaan, presensi, penilaian, setoran nadzom, & e-raport.
- **Cakupan Fitur**: Dashboard, Absensi Santri, Penilaian Nilai & Setoran Nadzom, Catatan Santri, Agenda, Pengumuman, E-Raport Kelas Binaan, Jadwal Mengajar, Profil.

### 2. Mufatish (Pimpinan Tingkatan / Pengawas)
- **Tanggung Jawab**: Audit & verifikasi tingkat madrasah.
- **Cakupan Fitur**: Monitoring Dashboard, Statistik Tingkatan, Monitoring Nilai, Monitoring Hafalan Nadzom, Monitoring Absensi, Monitoring Guru & Kelas, Pengumuman.

### 3. Mundzir (Pimpinan Madrasah / Pengasuh)
- **Tanggung Jawab**: Pengawasan eksekutif pimpinan pondok.
- **Cakupan Fitur**: Executive Monitoring Dashboard Global, Grafik Statistik, Monitoring Guru, Monitoring Seluruh Tingkatan, Monitoring Akademik, Hafalan & Kehadiran, Pengumuman.

### 4. Administrator (Software Admin Web Desktop)
- **Tanggung Jawab**: Pengelola Master Data & Sistem Administrasi Pesantren.
