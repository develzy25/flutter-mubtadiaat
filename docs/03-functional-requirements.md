# 03. Functional Requirements (FRD)

- FR-01: User dapat Login sesuai role (Mustahiq, Munawwib, Mufatish, Mundzir, Admin).
- FR-02: Mustahiq dapat mencatat absensi & mengelola e-raport kelas binaannya.
- FR-03: Munawwib dapat menginput nilai & hafalan mata pelajaran/kitab yang diampunya.
- FR-04: Mufatish dapat melakukan verifikasi akhir e-raport per tingkatan.
- FR-05: Mundzir dapat melihat laporan eksekutif dan statistik perkembangan santri.
